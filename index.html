Wallet - Projekt specjalnościowy

Aplikacja przeglądarkowa do zarządzania swoimi wydatkami

  • Wersja: 1.0
  • Autorzy: Dominik Chylo & Michał Borecki

Użytkownik

Funkcjonalności:

  1. Możliwość założenia konta
  2. Możliwość zalogowania się do swojego konta
  3. Po zalogowaniu powinny wyświetlić się mu jego portfele
    • te których jest właścicielem (utworzone przez tego użytkownika)
    • te do których należy (został zaproszony przez innego użytkownika)

Portfel

Funkcjonalności:

  1. Po wejściu do portfela użytkownikowi powinny wyświetlić się transakcje w nim dokonane

Kategoria

Funkcjonalności:

  1. Możliwość dodania kategorii do transakcji

Transakcja

Funkcjonalności:

  1. Możliwość wyświetlenia listy transakcji
  2. Możliwość dodania nowej transakcji
  3. Możliwość edycji transakcji
  4. Możliwość usunięcia transakcji

User

  • id int
  • name varchar(255)
  • email varchar(255)
  • password varchar(255)

Wallet

  • id int
  • name varchar(255)
  • owner_id int (user that created this wallet)

Category

  • id int
  • name varchar(255)
  • slug varchar(255)

Transaction

  • id int
  • wallet_id int
  • name varchar(255)
  • value bigint
  • type varchar(255) expense || income

value w bazie danych nie jest przechowywane w postaci zmiennoprzecinkowej. Wartość 35.79zł będzie przechowana jako 3579 w celu uniknięcia przekłamań przy działaniach na liczbach zmiennoprzecinkowych. Dopiero frontend przedstawi liczbę z dwoma liczbami po przecinku.
